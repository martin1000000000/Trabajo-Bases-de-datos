#  Trabajo Bases de Datos — Grupo 3

Proyecto del ramo de **Bases de Datos** que modela una plataforma de streaming de anime inspirada en **Crunchyroll**, implementada en **PostgreSQL**.

---

##  Descripción

El sistema simula la operación completa de un servicio de streaming de anime, incluyendo:

- **Gestión de usuarios** con planes de suscripción, perfiles múltiples y métodos de pago.
- **Catálogo de contenido** con películas y series anime (temporadas y episodios).
- **Interacciones** como visualizaciones, valoraciones, listas personalizadas y recomendaciones.
- **Datos sintéticos masivos** (~200k–400k filas) distribuidos en 4 años (2022–2025).

---
